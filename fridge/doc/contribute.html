How to Contribute a New HTML Page

  1. Write the page.
  2. Test the page by placing it on a publicly readable site, then see if it renders ok.
  3. Email the url of that page to mail@awk.info. Do NOT send the page.

      How to Contribute AWK Code

      • Zip the code up and email it to mail@awk.info.
      • Optionally, you can:
        • Add html into the code comments so it renders pretty on this site (for more details on that, see here).
        • Add a test suite to your code and, when this site renders your code, it include the test inputs/outputs.
      • E.g. if the page is http://menzies.us/tmp/xx.awk
      • Then try rendering it with http://awk.info/?awk:menzies.us/tmp/xx.awk
