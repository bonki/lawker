spell.awk

Purpose

  • Spell check files using user-defined dictionary and suffix files.
  • Show structuring methods for writing larger Awk programs.

Description

This program is an example par excellence of the power of awk. Yes, if written in "C", it would run faster. But this few lines of code implement a powerful spell checker implements a spelling check, with user-specifiable exception lists. The built-in dictionary is constructed from a list of standard Unix spelling dictionaries, overridable on the command line.

It also offer some tips on how to structure larger-than-ten-line awk programs. In the code below, note the:

  • moving much of the functionality out of PATTERN-ACTION and into functions;
  • how to parse command line options;
  • how to avoid excessive global variables (define most of the variables as local function variables;
  • Using the BEGIN block to initialize the system from files, and with local functions.
  • Using "print to pipes" to link the awk code to other commands.
